GNSS Fix status: Fix
Internal GNSS:
Disable if you do not want to use the router's internal GNSS receiver as a source for positioning data
GNSS Constellations:
Select which other satellite constellations are used to determine position,in addition to the GPS system
Differential positioning:
Activating differential positioning provides more accurate positioning in regions covered by Satellite Based Augmentation Systems(SBAS)
Differential positioning systems:
  • Alcomsat(Algeria)
  • AUS-NZ(Australia New Zealand)
  • BDSBAS(China)
  • EGNOS(Europe)
  • GAGAN(Indian)
  • KAAS(Korea)
