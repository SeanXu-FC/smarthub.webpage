Help

Click the links below to expand the available help content.

  • Status page
      The Status page provides information on the status of your router’s network connections, including the number of devices connected to the router’s access point (internal Wi-Fi) and the wired network (devices connected to the router’s 4 x RayNet network ports). Status information is provided for the following connections:
    • Cloud
      The Cloud status identifies when the Router has an active connection to the Raymarine cloud service.
    • Wi-Fi network
      The Wi-Fi network section shows the status of the external wireless network connection (e.g.: connection to marina Wi-Fi), the name of the connected network, and the signal strength.
    • Mobile network
      The Mobile network section shows the status of the mobile network connection, the name of your network provider, and which SIM card is in use.
    • Router access point
      The Router access point section shows the connection status of the internal Wi-Fi connection and the access point’s name (SSID).
  • Basic settings
      The following Basic settings pages are available:
    • Wi-Fi network page

        The router can connect to available Wi-Fi access points to provide internet access to the router and its connected devices. The Wi-Fi network page allows you to connect to a Wi-Fi access point, such as the Wi-Fi provided by your marina. When connected to a Wi-Fi connection that has a connection to the internet, the router will provide internet access to Wi-Fi and RayNet connected devices.

        Note: The router will connect automatically to saved Wi-Fi networks when they are in range. When available the Wi-Fi network connection will be used in preference to the cellular connection.

        From the Wi-Fi page you can:
      • enable and disable the Wi-Fi network
        Select the toggle switch to enable and disable the Wi-Fi network.
      • view details of available Wi-Fi networks that are in range
        Select the Info icon next to the Wi-Fi network to view details about the network.
      • connect to an available Wi-Fi network
        Follow the steps below to connect to a Wi-Fi network that is listed on the Wi-Fi network page.
        • 1. Select the Wi-Fi network you want to connect to from the list.
        • 2. Enter the password for the network in the Password field.
        • 3. Select Connect.
      • add a Wi-Fi network manually
        You can connect to a network that is not in the list but is in range, i.e.: a hidden Wi-Fi network (Wi-Fi network that is not broadcasting its SSID).
        • 1. Select Add network.
        • 2. Enter the Wi-Fi network name (SSID) in the Network name field.
        • 3. Select the security type from the Security drop down box.
        • 4. Enter the networks password in the Password field.
        • 5. Select Connect.
      • forget a saved Wi-Fi network
        When the router connects to a Wi-Fi network the details will automatically be saved so that the router can connect to it automatically when ever it is in range. If you do not wish to connect to this Wi-Fi network in future you can ‘forget’ the network by following the steps below:
        • 1. Select the Info icon next to the saved Wi-Fi network to display the network's information.
        • 2. Select Forget network.
          • Alternatively, you can also turn off automatic network connection by disabling Connect automatically from the Info dialog.
    • Mobile data & SIM management page

        The router can use cellular networks to provide internet access to the router and its connected devices. The Mobile data & SIM management page provides access to settings related to the router’s cellular connections, SIM management options and mobile data usage statistics. When the cellular connection is active and data is available the router will provide internet access to Wi-Fi and RayNet connected devices.

        Note: The Router will connect automatically to saved Wi-Fi networks when they are in range. When available the Wi-Fi network connection will be used in preference to the cellular connection.

        When using dual SIM cards you can enable automatic switching between SIMs when there is no network coverage or when data limit on one SIM has been reached, using the checkbox at the top of the page.

        From the Mobile data & SIM management page you can select SIM1 or SIM2 to view information and settings for each SIM card.

        The available information and options are:

        • Mobile data: — switches mobile data on and off, when switched on the router can access the internet using your SIM card’s data allowance.
        • Data roaming: — switches data roaming on and off. Data roaming allows you to use your data allowance when outside of your home country.
        • Data usage graph — view data usage statistics.
        • Data usage cycle — set the date at which your data allowance renews each month.
        • Data warning and limit — set data warning and limit so that you do not exceed your data allowance. You will be notified by the RayConnect app when the specified data warning and data limit for the month has been reached.
        • SIM info — View SIM card details.
        • Lock SIM — Lock the SIM card or change the pin for the SIM card.
    • Router access point page

        Wi-Fi enabled devices can connect to the router’s access point and connect to the internet using the router’s Wi-Fi network connection or cellular network connections. The Router access point page provides access to the router’s access point settings.

        The router’s access point can be enabled and disabled using the toggle switch at the top of the page. When enabled the router’s access point settings can be configured.

        The following settings are available:

        • Access point name (SSID) — this is the name of the network that you should connect your mobile device(s) to.
        • Password — this is the password that needs to be entered when connecting your mobile device(s).
        • Preferred Wi-Fi channel — allows you to select your preferred Wi-Fi channel. Interference can occur in areas where many Wi-Fi networks are using the same channe. Moving to a less-congested channel should eliminate this interference.
        • Channel width — allows you to switch between single channel (20 MHz) 144 Mb and Dual channel (40 MHz) 300 Mb channel width. Dual channel provides faster speed, however in areas where there is wireless channel congestion a single channel may reduce interference.
        • Encryption type — allows selection of the type of encryption used, WPA2–PSA is the default and recommended encryption type.

          Note: If the encryption type is changed to No encryption anyone within range will be able to connect to the router.

        If any of the default settings are changed, select Save at the top of the page to save the settings.
    • Info page
        Information about your router can be found on the Info page. The following information is available:
        • - Model name
        • - Model number
        • - Serial number
        • - IMEI number
        • - Voltage
        • - Current draw
        • - Temperature
        • - Operating hours
        • - Ethernet MAC addresses
        • - Wi-Fi MAC addresses
        • - Software version
          • - Application version
          • - Platform version
          • - Product bundle version
        • Cellular signal strength (RSSI)
        A QR code is available at the bottom of the Info page that can be used for onboarding to the Raymarine cloud service. For troubleshooting purposes you can also save any crash logs by selecting Save crash logs.
  • Connected devices page
    The connected devices page provides a list of all devices connected to the router using ethernet or wireless connections. The details available are:
    • - Device name
    • - MAC address
    • - IP address
    • - Connection type
  • Advanced settings
      The following Advanced settings pages are available:
    • Ethernet configuration page

      The Ethernet configuration page provides advanced settings for the SeaTalkhs(RayNet) wired network.

      By default and to ensure compatibility with Raymarine MFDs the Configure IP setting is set to Automatically (DHCP on). Under normal circumstances these settings should not need to be changed.

      The available Configure IP options are:

      • Automatically (DHCP on) — This is the recommended setting which allows IP addresses to be automatically assigned to connected devices by the router using IP addresses within the range stated in the DHCP server settings.
      • Manually (DHCP on) — This setting allows you to manually configure the router’s LAN IP address, subnet mask and default gateway and allows connecting devices to be automatically assigned an IP address within the range you specify in the DHCP server settings.
      • Manually (DHCP off) — This settings allows you to manually configure the router’s LAN IP Address, subnet mask and default gateway, but does not assign IP addresses to connected devices.

        Note: With DHCP switched off, each device will have to be manually assigned an IP address in the same range as your router’s IP address.

    • Wi-Fi configuration page

      The Wi-Fi configuration page provides advanced settings for the router’s Wi-Fi access point.

      The Router’s Wi-Fi IP address, subnet mask and default gateway can be configured. The IP address range used for the Wi-Fi DHCP server can also be set.

    • GNSS page
      The GNSS page provides settings and information for the router’s internal / built-in GNSS receiver.
      • GNSS fix status — provides position fix status.
      • Internal GNSS — enables and disables the internal GNSS receiver.
      • Restart GNSS — reboots the internal GNSS receiver.
      • GNSS Constellations — enables use of a second GNSS constellation (either GLONASS or Beidou GNSS can be used in addition to the GPS constellation).
      • Differential positioning — enables use of Satellite Based Augmentation Systems (SBAS) to enhance your position fix.
      • Differential positioning systems — allows selection of specific localized Satellite Based Augmentation Systems (SBAS).
      • Satellites in use — provides details of the positioning satellites currently being tracked.
    • Inputs & outputs
      Settings and status pages are available for the management of devices connected to the router’s inputs & outputs connections.
      • Channel monitoring and control page — view channel input and output details:
        • - For input channels (channels 1 to 4) the channel number, channel name and voltage is displayed.
        • - For output channels (channels 5 to 8) the channel number, channel name is displayed and a toggle switch to turn the channel on and off is available.
      • Channel configuration page — configure input and output channels:
        • - For input channels (channels 1 to 4) the channel name and input type can be configured. The input type can be configured as either: Analog, Digital positive or Digital negative.
        • - For output channels (channels 5 to 8) the channel name can be configured.
      • Alert notifications page — this page allows configuration of low voltage alert notifications. A product low voltage warning can be set which triggers when the voltage available to the router drops below a specified value. Low voltage warnings can also be configured for each input channel (Channels 1 to 4).

        Note: Each input channel warning must be configured with a unique Alert ID with a value between 1,000 and 65,000.

    • Power management page
      The router includes power management options to help reduce power consumption whilst maintaining the ability to connect to the router remotely.

      With Always on selected the router is fully powered.

      With Low power mode selected the router is placed into a low power state. In Low power mode Wi-Fi connections are disabled.

      In low power mode the router can be woken by:

      • - a remote device connected to the Raymarine cloud service, via the RayConnect app
      • - an alert notification, configured in the Alert notifications page.
      • - a Wake On LAN (WOL) signal from a device connected to the router’s SeaTalkhs network ports, when LAN signal is selected under Also wake On:.
      • - a device such as a switch, connected to one of the router’s input channels, when the relevant input channel is selected under Also wake On:.

      Note: When power cycled the router will also revert to Always on mode.

      The input channels (Channels 1 to 4) can also be used to place the router into low power mode.

      The router’s output channels (Channels 5 to 8) can be used to wake connected devices or systems that have a wake-on-power input feature. Place a tick in the relevant output channel to wake the connected device when the router is woken from lower power mode.

      Important: To keep changes made to the Power management options click Save.

    • Software upgrade page

      Raymarine regularly issues software updates for its products which provide new and enhanced features and improved performance and usability. It’s important to ensure that you have the latest software for your products by regularly checking the Raymarine website for new software releases.

      To upgrade the software on your router follow the steps below:

      Note: The instructions below are for upgrading using a personal computer

      • 1. Check your router’s current software version (Software version is available on the Software upgrade page).
      • 2. If available, Download updated software from the Raymarine website: www.raymarine.com/software
      • 3. Open the router’s Software upgrade page located under Advanced settings.
      • 4. Click Browse file to upload.
      • 5. Locate and select the downloaded file. The file will then be uploaded to the router.
      • 6. Click Upgrade. The router will now be upgraded. The upgrade process can take some time, do NOT disconnect the device you are upgrading from until the process is complete.

      Note: Once the process is complete the router will reboot.

    • Restart and factory reset page
      If you encounter problems with your router it can be restarted or reset to its factory default settings from the Restart & factory reset page.
      • - Click Restart router to reboot the router.
      • - Click Reset to factory defaults to remove all changes made to the router's settings and reset it to the default settings.
    • Admin password page

      The default password used to access the router’s web interface can be changed.

      To change the password follow the steps below:

      • 1. Enter the current password in the Current password: field.
      • 2. Enter your new password in the Create new password: field.
      • 3. Re-enter the new password in the Confirm new password: field.
      • 4. Click Change password.
