  • SET-UP WIZARD
      The Set-up wizard page guides you through the necessary settings to set up your wireless network connections.
    • Wireless network (external WLAN)
      The first page allows you to connect to a wireless network, such as the WLAN provided by your marina. This will enable the Router and devices connected to it to connect to the internet.
    • Router access point (internal WLAN)
      The second page allows you to configure the Router access point's name and password. These are the details required to connect your mobile devices to the Router.
    • More information
  • BASIC SETTINGS
      The Basic settings provides access to settings for WLAN, Mobile data (Cellular) and the Router’s access point. The Basic settings options also provide access to product information.
    • Wireless network (external WLAN)
      • WLAN switch
        Switches the Router’s external WLAN on and off. When switched off the Router will be unable to connect to an external WLAN.
      • List of available wireless networks
        Connect to an available wireless network, such as the WLAN provided by your marina. This will enable the Router and devices connected to it to connect to the internet. You can also connect to a network manually by selecting Add network and entering that network's connection details.
    • Mobile data (cellular)
      • Mobile data switch
        Switches mobile data on and off, when switched on the Router can access the internet using your SIM card’s data allowance.
      • Data roaming switch
        Switches data roaming on and off. Data roaming allows you to use your data allowance when outside of your home country.
      • Advanced > Data usage
        Allows you to view data usage statistics and set data limits for your mobile data.
      • Advanced > SIM management
        Allows you to select which SIM card is in use and to assign rules for automatically switching between SIM cards.
      • SIM1 / SIM2
        By selecting SIM1 or SIM2 you can view details for your SIM card(s) and set up SIM locks.
    • Router access point (internal WLAN)
        The Router access point page allows you to set up the access point’s connection settings so that your wireless devices can connect to the Router and use its internet connection.
      • Access point switch
        Switches the Router’s internal WLAN on and off. When switched off wireless devices will not be able to connect to the Router.
      • Access point connection settings
        • Access point name — This is the name of the network that you should connect your wireless device(s) to.
        • Password — This is the password that needs to be entered when connecting your wireless device(s).
        • Channel width — Allows you to switch between single channel (20 MHz) 144 M/bit and Dual channel (40MHz) 300 M/bit. Dual channel provides faster speed, however in areas where there is high wireless congestion using a single channel may reduce interference.
        • Preferred WLAN channel — Allows you to select your preferred WLAN channel. Interference can occur in areas where many WLAN networks are using the same channel; moving to a less used channel should eliminate this interference.
        • Encryption type — Allows selection of the type of encryption used; WPA2–PSA is the default and recommended encryption type.
        Note: If the encryption type is changed to No encryption, anyone within range will be able to connect to your Router, and the devices connected to it. They will also be able to use your mobile data allowance.

  • ADVANCED SETTINGS
    • LAN configuration
      Provides settings for the wired (RayNet) network.
      Note: For correct operation with Raymarine MFDs ensure that Configure IP is set to 'Automatically (DHCP on)'.
      The following options are avaialble:
      • Automatically (DHCP on) — This is the recommended setting, this allows IP addresses to be automatically assigned to connected devices by the Router using IP addresses within the range specified in the DHCP server section.
      • Manually (DHCP on) — This setting allows you to manually configure the Router’s LAN IP address, subnet mask and default gateway and allows connected devices to be automatically assigned an IP address within the range you specify in the DHCP server section.
      • Manually (DHCP off) — This setting allows you to manually configure the Router’s LAN IP Address, subnet mask and default gateway, but does not assign IP addresses to connected devices.
      Note: With DHCP switched off, each device will have to be manually assigned an IP address in the same range as your Router’s IP address.
    • WLAN configuration
      Provides settings for the wireless (internal WLAN) network.
      The following options are available:
      • Manually — This setting allows you to manually configure the Router’s WLAN IP address, subnet mask and default gateway and specify the IP address range available.
      • Automatically — This is the recommended setting; it allows IP addresses to be automatically assigned to wireless devices connecting to the Router.
    • GNSS
      Provides settings related to the Router’s internal GNSS (GPS) receiver.
      • GNSS fix status — Provides position fix status.
      • Internal GNSS — Switches the internal receiver on and off.
      • Restart GNSS — Reboots the internal GNSS receiver.
      • GNSS Constellations — Enables use of GLONASS and Beidou GNSS constellations (in addition to the GPS constellation).
      • Differential positioning — Enables use of differential positioning satellites which enhance your position fix.
      • Differential positioning systems — Allows selection of specific differential systems.
      • Satellites in use — Provides details of the satellites being used for your position fix.
    • Inputs and outputs
      Allows management of devices connected to the Router’s input and output connections.
      • Channel monitoring — This page provides monitoring details of connected input and output devices.
      • Channel configuration — This page allows configuration of connected input and output devices.
      • Alert notifications — This page allows configuration of alert notifications for connected input and output devices.
    • Software upgrade
      Upload and install new Router software.

      Your Router's current software version can be found on the Info page (BASIC SETTINGS > Info). To search for updated software for your Router, go to the software updates section of the Raymarine website: Software updates
